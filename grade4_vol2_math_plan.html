第1单元 · 建议学习：4天（超重要！）
四则运算（运算顺序）
掌握加减乘除混合运算的顺序规则，含括号的运算，理解0和1的特殊运算
📖

核心知识点

  • 运算顺序口诀（必须背！） ①同级运算（只有加减，或只有乘除）：从左到右依次计算
    ②不同级：先乘除，后加减
    ③有括号：先算括号里面的（小括号→中括号→大括号）
  • 加法和减法是第一级运算 只有加减时，从左到右算。如：30-12+8 = 18+8 = 26（不能先算12+8）
  • 乘法和除法是第二级运算 加减混乘除时，先算乘除。如：3+4×5 = 3+20 = 23（不能先算3+4）
  • 括号改变运算顺序 小括号内先算。如：(3+4)×5 = 7×5 = 35（括号改变了顺序）
  • 0和1的特殊性质 a+0=a；a-0=a；a×0=0；a×1=a；a÷1=a；0÷a=0（a≠0）；a÷0无意义！
⚡ 运算顺序速记
先括号，再乘除，最后加减，同级从左到右
小括号 → 中括号 → 大括号（由内到外）
最容易错："3+4×5"中很多孩子先算3+4=7，这是错的！没有括号时，乘除先于加减。做混合运算前，先把要先算的那步画圈标记，再计算。

第3单元 · 建议学习：4天（非常重要！）
运算定律与简便运算
掌握加法和乘法的五大运算定律，灵活运用进行简便计算
📖

核心知识点——五大运算定律

  • ① 加法交换律 a + b = b + a   例：36+64 = 64+36 = 100
  • ② 加法结合律 (a+b)+c = a+(b+c)   例：37+46+54 = 37+(46+54) = 37+100 = 137
  • ③ 乘法交换律 a × b = b × a   例：25×4 = 4×25 = 100
  • ④ 乘法结合律 (a×b)×c = a×(b×c)   例：25×4×9 = (25×4)×9 = 100×9 = 900
  • ⑤ 乘法分配律（最重要！） (a+b)×c = a×c + b×c   例：(25+4)×4 = 25×4+4×4 = 100+16 = 116
    逆用：36×101 = 36×(100+1) = 36×100+36×1 = 3600+36 = 3636
⚡ 简便计算常用凑整组合
25×4=100 125×8=1000 50×2=100
凑整思路：找能组成整十/整百/整千的数对
乘法分配律是难点：正用——展开括号；逆用——把公因数提出来。如 36×99+36 = 36×(99+1) = 36×100 = 3600。逆用需要多练，非常考验灵活性。
✏️

小数的意义和性质
深入理解小数的意义，掌握小数的读写、大小比较、性质及单位换算
📖

核心知识点

  • 小数的意义 把1平均分成10份，每份是0.1（十分之一）；分成100份，每份是0.01（百分之一）；分成1000份，每份是0.001（千分之一）
  • 小数各数位名称 小数点左边：个位、十位、百位……小数点右边：十分位、百分位、千分位……
    如3.456：3在个位，4在十分位，5在百分位，6在千分位
  • 小数的性质（重要！） 小数末尾添上0或去掉0，小数的大小不变。如：0.8=0.80=0.800；3.50=3.5
    注意：中间的0不能去！如3.05≠3.5
  • 小数大小比较 从整数部分开始，依次比较各位数字，第一个不同的数位上数字大的那个小数大。
  • 小数点的移动规律 小数点右移：×10（移1位）、×100（移2位）、×1000（移3位）
    小数点左移：÷10（移1位）、÷100（移2位）、÷1000（移3位）
  • 小数与单位换算 3.5米=350厘米；0.85千克=850克；1.2千米=1200米。关键：单位变小→数变大→小数点右移
⚡ 小数点移动规律
右移1位 ×10，右移2位 ×100，右移3位 ×1000
左移1位 ÷10，左移2位 ÷100，左移3位 ÷1000
末尾添/去0，大小不变（小数性质）
易错陷阱：①0.8和0.80大小相同（末尾添0不变大小），但0.8和0.08不同（中间的0不能省）②比较大小时，0.9和0.19，有人误认为0.19大（位数多），实际0.9>0.19
✏️

第5单元 · 建议学习：4天
三角形
认识三角形，掌握三角形的分类与特性，学习内角和及三边关系
📖

核心知识点

  • 三角形的定义 由三条线段围成的封闭图形。三角形具有稳定性（不容易变形），如自行车架、屋顶桁架。
  • 三角形内角和 = 180°（必须记！） 任意三角形的三个内角之和都等于180°，可以用来求未知角。
  • 三边关系 两边之和 > 第三边；两边之差 < 第三边。如：3cm、4cm、8cm不能围成三角形（3+4=7<8）
  • 按角分类 锐角三角形：三个角都是锐角；直角三角形：有一个直角；钝角三角形：有一个钝角
  • 按边分类 等边三角形：三边相等，三角均60°；等腰三角形：两边相等，两底角相等；不等边三角形：三边都不等
  • 三角形的高 从一个顶点向对边作垂线，垂线段叫做这条边上的高，对应的边叫做底。每个三角形有三条高。
⚡ 关键公式
三角形内角和 = 180°
两边之和 > 第三边（三边关系）
等边三角形：每个角 = 60°
等腰三角形：顶角+底角×2 = 180°

小数的加法和减法
掌握小数加减法的计算方法，学会估算和验算，用小数解决实际问题
📖

核心知识点

  • 小数加减法的计算法则 ①小数点对齐（即相同数位对齐）②从最低位加起（或减起）③处理进位/退位④结果的小数点与上面对齐⑤末尾多余的0可以去掉
  • 位数不同时补0 计算时可在末尾补0使位数相同，便于对齐。如：3.6+1.45，把3.6写成3.60再计算
  • 估算（先估再算，验证合理性） 把小数四舍五入到整数进行估算。如3.87+2.15≈4+2=6，精确值5.02偏小了，再检查。
  • 整数减小数 如5-1.36，把5写成5.00，然后按小数减法计算：5.00-1.36=3.64
⚡ 计算要点
① 小数点对齐（最关键！）
② 位数不同→末尾补0→再计算
③ 整数-小数→整数后加".00"
最常见错误：小数点没对齐！比如计算3.6+12.45时，把3.6的6和12.45的5对齐（当成整数），导致计算完全错误。每次下笔前先找好小数点位置。

图形的运动（二）
认识轴对称图形，学习图形的旋转，理解平移与旋转的区别
📖

核心知识点

  • 轴对称图形 沿一条直线折叠，两边完全重合的图形。这条直线叫对称轴。正方形有4条，长方形有2条，圆有无数条。
  • 旋转的三要素 ①旋转中心（绕哪个点转）②旋转方向（顺时针/逆时针）③旋转角度（转了多少度）
  • 旋转的性质 旋转后图形的大小、形状不变；对应点到旋转中心的距离相等；对应线段的长度不变。
  • 常见旋转角度 顺时针旋转90°=向右转；逆时针旋转90°=向左转；旋转180°=翻转到对面；旋转360°=回到原处
生活中的旋转：风扇叶片、钟表指针、陀螺、旋转木马——都是绕固定点旋转。讨论生活中的旋转例子，帮助孩子建立旋转的直觉。

第8单元 · 建议学习：3天
平均数与条形统计图
理解平均数的意义和求法，学会用复式条形统计图分析数据
📖

核心知识点

  • 平均数的意义 平均数是一组数据的"代表值"，表示这组数据的整体水平。它不一定是数据中实际存在的某个值，可以是小数。
  • 平均数的计算 平均数 = 所有数据之和 ÷ 数据个数。如：4个数10、8、12、14，平均数=(10+8+12+14)÷4=44÷4=11
  • 平均数的范围 平均数一定在最小值和最大值之间。可以用来判断计算是否合理。
  • 复式条形统计图 用不同颜色（或花纹）的条形在同一图中表示多组数据，便于比较不同类别之间的差异。必须看懂图例。
⚡ 平均数公式
平均数 = 总和 ÷ 个数
总和 = 平均数 × 个数（逆推）
最小值 < 平均数 < 最大值
